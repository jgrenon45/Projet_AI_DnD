
# Le Grimoire du Maitre du Donjon

Assistant IA pour Dungeons & Dragons 5e avec interface retro 8-bit.

## Membre Equipe
Philip Jolin (JOLP12040300)
Edouard Martel ()
Jeremy Grenon ()

# Prerequis 
Avoir Ollama et LM studio en marche avec un model LLM en marche

## LM Studio
1. Telecharger depuis https://lmstudio.ai
2. Charger un modele (Qwen, Llama, Mistral...)
3. Demarrer le serveur local (port 1234)

## Ollama
1. Installer depuis https://ollama.ai
2. ollama pull llama2

# Installation
pip install -r requirements.txt

# Lancer application
python run.py
